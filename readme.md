# void-elem-ui

[![npm version](https://img.shields.io/npm/v/void-elem-ui.svg?style=flat-square)](https://www.npmjs.com/package/void-elem-ui)
[![license](https://img.shields.io/badge/license-MIT-blue.svg?style=flat-square)](LICENSE)

基于 Element UI 二次封装的 Vue 2.6 组件库，在原生 Element UI 基础上优化交互体验、补充业务通用组件，降低业务开发成本。

## 特性

- 🎯 完全兼容 Element UI 原有 API，无侵入式升级
- 🛠️ 封装高频业务组件（如搜索表单、数据表格、操作弹窗等）
- 🎨 统一的视觉风格和交互规范，可自定义主题
- ⚡ 轻量化封装，无冗余代码，保持原有的性能表现
- 📖 完善的使用示例，快速接入业务开发

## 安装

### npm 安装

```bash
# 先安装 Element UI（核心依赖）
npm i element-ui@^2.15.13 -S

# 安装本组件库
npm i void-elem-ui -S
```
